# homebrew-tap

Homebrew tap for installing tools published by [Tendo33](https://github.com/Tendo33).

## Install

Tap the repository first:

```bash
brew tap Tendo33/homebrew-tap
```

Then install a cask:

```bash
brew install --cask iconkit
```

Or install directly in one command:

```bash
brew install --cask Tendo33/homebrew-tap/iconkit
```

## Available Casks

- `iconkit` - Developer-friendly CLI tool for icon processing.

## Upgrade

```bash
brew update
brew upgrade --cask iconkit
```

## Uninstall

```bash
brew uninstall --cask iconkit
```

If you no longer need this tap:

```bash
brew untap Tendo33/homebrew-tap
```

## Maintenance

Casks in this repository are updated automatically from their source repositories during release.

For generated casks, manual edits may be overwritten by the next automated release.

## Source Repositories

- `iconkit`: [Tendo33/iconkit](https://github.com/Tendo33/iconkit)
- Releases: [https://github.com/Tendo33/iconkit/releases](https://github.com/Tendo33/iconkit/releases)
